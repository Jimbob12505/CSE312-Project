## Setup Instructions
- create a new venv by ```python3 -m venv venv```
- activate venv by doing ```source venv/bin/activate```
- install requirements: ```pip install -r requirements.txt```
- cd into frontend
- run ```npm run build```
- run ```npm run dev```
- deactivate venv by ```deactivate```
